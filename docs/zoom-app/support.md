Support – dlzoom

If you get stuck, we’re happy to help.

How to get help
- GitHub: open an issue with what happened and what you expected.
- Email: yaniv+dlzoom@golan.name

Helpful details to include
- Your OS (macOS/Windows/Linux) and dlzoom version.
- The command you ran and the full output (remove any tokens).
- Whether you used the hosted sign‑in (when enabled) or your own.

Security
- If you think you found a security issue, email us at yaniv+dlzoom@golan.name. Don’t post details publicly. We aim to acknowledge within 72 hours.

Service level
- dlzoom is a community, open‑source project with no SLA. We provide best‑effort support.

Check service status
- Using the hosted sign‑in (when enabled)? We’ll note incidents on GitHub issues.
- Hosting your own sign‑in? Check your cloud provider’s status/dashboard.

Uninstall / revoke access
- In Zoom: App Marketplace → Manage → Added Apps → Remove (or Profile → Apps → Manage → Remove).
